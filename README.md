Sistema simples em React para gerencimento de tarefas.

## Start

- NodeJS e npm versions `React 16.8.6`

- Instalar dependências: `npm install` or `yarn`

- Start server: `npm run start` or `yarn start`

- Acessar: `localhost:3000`
